# Texo


### Ambiente

Projeto desenvolvido utilizando as tecnologias:

```
Java 8, devTools, H2 database, OpenCSV
```
### Execução

* **Com a IDE escolhida execute a aplicação**

* **Acesse** - http://localhost:8080/awards

### Execução (Por linha de comando)

* **Acesse o diretório raiz do projeto e execute em linha de comando:**
* mvnw clean
* mvnw install
* Copie o arquivo movielist.csv para o diretorio target

* **por linha de comando no diretório target execute:**
* java -jar texo-0.0.1-SNAPSHOT.jar


* **Acesse** - http://localhost:8080/awards
